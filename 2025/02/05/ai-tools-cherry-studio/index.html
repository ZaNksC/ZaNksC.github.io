一站式AI工作台
发布于:
2025-02-05
更新于:
2025-02-06
作者:
Za_Nks
#Deepseek #硅基流动 #Cherry studio

大模型浪潮下，生产力工具被颠覆性革命，顺应潮流，给自己打造一个趁手的AI工作台

前言

随着2025年过年期间DeepSeek模型的火爆出圈，AI在这个春节进入千家万户，安装DeepSeek成为了潮流。但DeepSeek的官网也在恶意进攻（存疑）和群众火爆的热情中间歇性瘫痪。在这样的背景下，硅基流动与华为云合作部署的DeepSeek R1模型与本地部署DeepSeek成为了新的潮流解决方案。为了赶时髦，我也体验了一番硅基流动的服务，虽然R1和V3卡顿得让我觉得体验不佳，但我却发现了CherryStudio这个好玩的一站式的AI工作台。

安装

硅基流动

硅基流动是一家AI的服务提供商，提供按需付费的AI服务。目前注册或邀请都可以获得 2000W Tokens的配额（14元赠送金） 。可以点击我的邀请链接，帮我获得2000W Tokens，也可以直接前往官网注册。

注册成功后，可以看到非常多的模型，其中就包括了最近大火的deepseek（R1和V3）

开始使用前，需要完成实名认证。之后可以直接在硅基流动的平台上使用，也可以通过其他客户端。如果需要通过其他客户端使用的话，需要创建API密钥。点击左侧的API密钥菜单后，点击新建API密钥，随便输入一个描述即可。

创建成功后，点击密钥就可以复制，后面配置客户端的时候会用到。

硅基流动还有一个体验非常好的地方就是账单的速度很快，体验了一下就知道大概会花多少钱了。

最后推荐几个使用过的模型：

模型 评价 费用
deepseek-ai/DeepSeek-R1 最近非常火的深度思考模型，但我使用下来速度过慢，体验不是很好，实际使用价格偏高 输入 ¥4 / 1M Tokens，输出 ¥16 / 1M Tokens
deepseek-ai/DeepSeek-V3 最近非常火的深度思考模型，但我使用下来速度过慢，体验不是很好，实际使用价格偏高 输入 ¥2 / 1M Tokens，输出 ¥8 / 1M Tokens
deepseek-ai/DeepSeek-V2.5 体验优秀，价格合适，非常适合作为需要一定思考强度的模型使用 ¥1.33 / 1M Tokens

meta-llama/Meta-Llama-3.1-8B-Instruct 用来做搜索比较好用，速度不慢，翻译也OK 免费

Cherry Studio

Cherry Studio是一个支持多服务商集成的AI对话客户端，提供了以API方式接入大部分模型的能力。

可以从官网下载对应版本并安装。

安装完成后，在设置页面配置硅基流动的API（配置这一家就差不多了，其他的供应商不好白嫖，硅基流动的模型已经很多了）

在API密钥中填入硅基流动平台复制出来的密钥，点击检查，联通后可以开始使用。

左侧菜单栏中，毕竟常用的是对话、图片、翻译。比如我的配置

  • 对话，安装收费情况配置了不同的模型，一般搜索用前两个，具有一定难度的使用其他模型（但其实不如各个服务商自己的网页服务，适用于一般搜索场景）
  • 图片，一般使用两个免费的模型，按需使用其他收费模型（对我这种轻度使用者来说区别不大）
  • 翻译界面没有其他配置，但是在默认模型中修改了翻译服务的模型（其实用收费模型就可以了，翻译用的tokens很少）

另外，我还会使用CherryStudio中的小程序来直接使用豆包和Kimi的对话服务（但是CherryStudio这个做的不是很好，页面切换之后重新进入小程序会回到首页，上下文需要手动去找）

基于CherryStudio作为本地客户端，硅基流动提供云端AI服务，可以非常简单的打造一个用于支持日常工作的一站式工作台，但是只能轻度使用，如果是图片处理相关的能力，需要ComfyUI支持，目前CherryStudio缺失了这块能力，我也没找到价格合适的平台（算力租赁价格对于非从业者来说还是偏高）。
